Getting Started

  • Set your default currency during onboarding or in Settings.
  • Add accounts and categories in Settings > Ledger.
  • Use Chat to record expenses, income, transfers, and recurring bills.
  • Manual entry is available when you prefer not to use AI.

AI Provider Setup

AI MoneyPal supports user-configured OpenAI-compatible model providers. Enter your Base URL, API key, and model name in Settings > AI. API keys are stored locally in Apple's Keychain.

iCloud Sync

iCloud sync can be enabled in Settings > iCloud. It uses your private iCloud account through Apple's CloudKit service. If sync does not appear immediately on another device, confirm both devices use the same Apple ID and that iCloud Drive is enabled.

Receipts

Receipt images can be attached from Chat or from transaction editing. Vision-capable AI models can read receipt images directly. Text-only models can still use OCR text when available.

Import and Export

AI MoneyPal supports JSON backup, encrypted backup, and transaction CSV export. Keep encrypted backup passwords safe because they cannot be recovered by the app.

Privacy

AI MoneyPal is local-first. Ledger data stays on your device unless you enable iCloud sync or send selected content to your configured AI provider. Read the Privacy Policy for details.
